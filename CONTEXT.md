# LanFlow

LanFlow จัดการงานการเงินและการปฏิบัติงานระหว่างสาขา โดยใช้คำศัพท์ร่วมกันเพื่อแยกความหมายทางธุรกิจที่ใกล้เคียงกันให้ชัดเจน

## การแจ้งเตือน

**สรุป Badge ทาง Telegram**:
ข้อความเตือนตามรอบเวลาที่รวบรวมชื่อ Badge หรือสถานะส่วนกลางที่ยังรอ ค้าง หรือยังต้องจัดการและถูกเปิดใช้ พร้อมจำนวนและสาขาต้นทาง โดยไม่รวมสถานะที่สำเร็จหรือปิดงานแล้ว ไม่เปิดเผยรายละเอียดของแต่ละรายการ และไม่ส่งข้อความเมื่อไม่มีงานค้างที่ต้องรายงาน สถานะที่อยู่เฉพาะบนอุปกรณ์ไม่ถือเป็นส่วนหนึ่งของสรุปนี้
_Avoid_: แจ้งเตือนรายการใหม่, รายงาน Telegram

## การโยกเงินระหว่างสาขา

**การโยกเงินระหว่างสาขา**:
การนำเงินจากสาขาต้นทางไปยังสาขาปลายทาง โดยมีวิธีโยกเงินเป็นเงินสดหรือโอนธนาคาร
_Avoid_: โอนให้สาขา

**วิธีโยกเงิน**:
ช่องทางที่ใช้ส่งมอบเงินในการโยกเงินระหว่างสาขา มีสองแบบคือเงินสดและโอนธนาคาร
_Avoid_: ประเภทการโอน

**เงินสด**:
วิธีโยกเงินที่ส่งมอบเงินในรูปเหรียญและธนบัตรจริงจากสาขาต้นทางไปยังสาขาปลายทาง

**โอนธนาคาร**:
วิธีโยกเงินที่ส่งเงินผ่านบัญชีธนาคารและมีหลักฐานการโอน
_Avoid_: เงินโอน

**รอรับเงิน**:
สถานะของการโยกเงินสดที่สาขาต้นทางบันทึกการส่งเงินแล้ว แต่สาขาปลายทางยังไม่ได้ยืนยันว่าได้รับเงิน
_Avoid_: รอโอน, ค้างจ่าย

**ยืนยันรับเงิน**:
การที่สาขาปลายทางรับรองว่าได้รับเงินสดจากการโยกเงินระหว่างสาขาแล้ว
_Avoid_: อนุมัติรายการ

**จำนวนที่ส่ง**:
จำนวนเหรียญและธนบัตรที่สาขาต้นทางนับและส่งไปกับการโยกเงินสด

**จำนวนที่รับจริง**:
จำนวนเหรียญและธนบัตรที่สาขาปลายทางนับได้เมื่อรับการโยกเงินสด
_Avoid_: จำนวนที่รับ

**ผลต่างการรับเงิน**:
ส่วนต่างระหว่างจำนวนที่รับจริงกับจำนวนที่ส่ง แสดงได้ทั้งรายชนิดเงินและยอดรวม
_Avoid_: ยอดคงเหลือ

**ยอดไม่ตรง**:
สถานะการตรวจรับเงินสดที่จำนวนที่รับจริงไม่เท่ากับจำนวนที่ส่ง โดยต้องแสดงยอดผลต่างควบคู่กับสถานะ
_Avoid_: รับเงินแล้ว, เงินหาย

**ยอมรับผลต่าง**:
การที่ผู้มีสิทธิ์รับรองผลต่างการรับเงินและยุติการติดตามรายการ โดยคงจำนวนที่ส่งและจำนวนที่รับจริงเดิมไว้เป็นหลักฐาน
_Avoid_: ปิดยอดไม่ตรง, แก้ยอด, อนุมัติยอด

**ลบถาวร**:
การนำรายการโยกเงินและข้อมูลการตรวจรับที่เกี่ยวข้องออกจากระบบโดยไม่สามารถกู้คืนได้ ทำได้เฉพาะ super admin
_Avoid_: ยกเลิก, ลบรายการ

## การอนุมัติเปลี่ยนแปลงบิลยาง

**เวลาเริ่มนับการเปลี่ยนแปลงบิลยาง**:
เวลาที่บิลยางถูกบันทึกเข้าสู่ระบบส่วนกลางสำเร็จเป็นครั้งแรก ใช้เป็นจุดเริ่มต้นของช่วงเวลาที่เปลี่ยนแปลงได้โดยไม่ต้องขออนุมัติ
_Avoid_: เวลาออกบิลบนเครื่อง, วันที่ในบิล, เวลาซิงก์ล่าสุด

**ช่วงเวลาเปลี่ยนแปลงบิลยางโดยไม่ต้องอนุมัติ**:
ระยะเวลาหลังเวลาเริ่มนับการเปลี่ยนแปลงบิลยางที่ผู้มีสิทธิ์แก้ไขหรือลบบิลได้ตามปกติ เมื่อพ้นช่วงนี้ การเปลี่ยนแปลงที่อยู่ในขอบเขตควบคุมต้องได้รับอนุมัติก่อน
_Avoid_: เวลาหมดอายุบิล, เวลาล็อกบิล

**การแก้ไขบิลยางภายใต้การอนุมัติ**:
การเปลี่ยนข้อมูลธุรกิจใด ๆ ของบิลยางหลังพ้นช่วงเวลาเปลี่ยนแปลงโดยไม่ต้องอนุมัติ ครอบคลุมข้อมูลลูกค้า วันที่ น้ำหนัก ราคา รายการหัก และการจ่าย แต่ไม่รวมการเปลี่ยนสถานะการพิมพ์
_Avoid_: การอนุมัติเฉพาะราคา, การอนุมัติสถานะพิมพ์

**ราคายางที่กำหนด**:
ราคาต่อกิโลกรัมค่ากลางหนึ่งค่าที่ใช้เหมือนกันทุกสาขาและใช้ตรวจราคาของทุกรายการชั่งเมื่อสร้างบิลหรือเปลี่ยนราคาแบบจำนวนเงินสองตำแหน่งโดยไม่มีช่วงเผื่อ กฎอนุมัติราคาทำงานเมื่อมีค่านี้และหยุดทำงานเมื่อผู้มีสิทธิ์ล้างค่า
_Avoid_: ราคาเฉลี่ยยาง, ราคาบิล

**การอนุมัติราคาบิลยาง**:
การรับรองบิลใหม่หรือการเปลี่ยนราคาของรายการชั่งที่ใช้ราคาไม่ตรงกับราคายางที่กำหนด กฎนี้ไม่ขึ้นกับช่วงเวลาเปลี่ยนแปลงบิลยางโดยไม่ต้องอนุมัติ
_Avoid_: การอนุมัติราคาเฉลี่ย, การอนุมัติหลังหมดเวลา

**คำขอสร้างบิลยาง**:
ข้อเสนอข้อมูลบิลยางใหม่ทั้งใบที่มีรายการชั่งอย่างน้อยหนึ่งรายการใช้ราคาไม่ตรงกับราคายางที่กำหนด คำขอนี้ยังไม่ใช่บิลจริงและไม่เกี่ยวข้องกับรับ-จ่าย รายงาน รายการโอนเงิน หรือการส่งออกยางจนกว่าจะได้รับอนุมัติ
_Avoid_: บิลยางรออนุมัติ, บิลจริงสถานะรอ

**คำขอเปลี่ยนแปลงบิลยาง**:
ข้อเสนอให้แก้ข้อมูลธุรกิจหรือลบบิลยางหลังพ้นช่วงเวลาเปลี่ยนแปลงโดยไม่ต้องอนุมัติ โดยบิลจริงยังไม่เปลี่ยนจนกว่าคำขอจะได้รับอนุมัติ
_Avoid_: คำขอแก้ไขบิลยาง

**บิลยางรออนุมัติการเปลี่ยนแปลง**:
บิลยางที่มีคำขอเปลี่ยนแปลงหนึ่งรายการซึ่งยังไม่ถูกอนุมัติหรือลบ บิลนี้ยังคงใช้ข้อมูลเดิม เปลี่ยนแปลงเพิ่มไม่ได้ และยังนำเข้ารายงานหรือรายการโอนเงินไม่ได้ ส่วนบิลที่อยู่ในรายงานหรือรายการโอนเงินแล้วจะสร้างคำขอเปลี่ยนแปลงไม่ได้
_Avoid_: บิลฉบับแก้ไข, บิลร่าง

**ผู้อนุมัติบิลยาง**:
ผู้ใช้ที่เป็น super admin หรือได้รับสิทธิ์ผู้จัดการระบบ มีสิทธิ์อนุมัติหรือลบคำขอสร้างและคำขอเปลี่ยนแปลงบิลยางของทุกสาขา ผู้อนุมัติยังต้องสร้างคำขอเมื่อข้อมูลของตนเข้าเงื่อนไข แต่อนุมัติคำขอนั้นเองได้
_Avoid_: ผู้ดูแลสาขา, ผู้สร้างบิล

**การลบคำขอบิลยาง**:
การนำคำขอสร้างหรือคำขอเปลี่ยนแปลงที่ยังรออนุมัติออกจากระบบอย่างถาวรโดยไม่สร้างหรือเปลี่ยนบิลจริง ทำได้เฉพาะผู้อนุมัติบิลยาง ส่วนคำขอที่อนุมัติแล้วลบไม่ได้
_Avoid_: ปฏิเสธคำขอ, ยกเลิกคำขอ

**การลบบิลยางภายใต้การอนุมัติ**:
การทำให้บิลยางพ้นช่วงเวลาเปลี่ยนแปลงหยุดใช้งานหลังคำขอลบได้รับอนุมัติ โดยยังเก็บบิล ผู้ลบ และเวลาลบไว้เป็นหลักฐาน
_Avoid_: ลบคำขอเปลี่ยนแปลง, ลบถาวร

**ระยะเวลาอนุญาตเปลี่ยนแปลงบิลยาง**:
จำนวนนาทีเต็มค่ากลางหนึ่งค่าที่กำหนดช่วงเวลาเปลี่ยนแปลงบิลยางโดยไม่ต้องอนุมัติ และใช้เหมือนกันกับทุกสาขา ผู้มีสิทธิ์ผู้จัดการระบบเป็นผู้กำหนดค่านี้ โดยค่า 0 หมายถึงต้องขออนุมัติทุกครั้งหลังบันทึกบิลเข้าสู่ระบบส่วนกลางครั้งแรก
_Avoid_: เวลาเปลี่ยนแปลงรายสาขา, เวลาเปลี่ยนแปลงรายผู้ใช้

## การส่งออกยาง

**รายการส่งออกยาง**:
ชุดบิลยางที่ผ่านการล็อกจากรายงานแล้วและถูกรวบรวมเพื่อคำนวณน้ำหนัก ต้นทุน และค่าทำงานของการส่งออกหนึ่งครั้ง
_Avoid_: รายงานยาง, บิลส่งออก

**เวลาพร้อมออกรายงาน**:
เวลาที่รายการต้นทางมีคุณสมบัติพร้อมถูกนำเข้ารายงาน ใช้เป็นลำดับเวลาสำหรับกำหนด cutoff ของรายการส่งออกยาง
_Avoid_: เวลาสร้างบิล, เวลาพิมพ์รายงาน

**ตรวจสอบแล้ว**:
สถานะปิดงานของรายการส่งออกยางที่ผู้มีสิทธิ์รับรองว่าน้ำหนักปัจจุบันและข้อมูลค่าทำงานถูกต้องครบถ้วนแล้ว
_Avoid_: อนุมัติรายงาน, จ่ายแล้ว

**น้ำหนักสุทธิหลังหัก**:
น้ำหนักยางสุทธิจากการชั่งของบิลยางหลังหักน้ำหนักยางเพิ่มเติมที่ระบุในบิลแล้ว
_Avoid_: น้ำหนักสุทธิจากการชั่ง, น้ำหนักปัจจุบัน

**รายจ่ายจากรายการส่งออกยาง**:
มุมมองรายจ่ายแบบอ่านอย่างเดียวที่เกิดจากรายการส่งออกยางซึ่งตรวจสอบแล้ว โดยรายการส่งออกยางยังคงเป็นข้อมูลต้นทางเพียงรายการเดียว
_Avoid_: บิลรายจ่ายส่งออกยาง, สำเนารายจ่าย

**น้ำหนักปัจจุบัน**:
น้ำหนักยางที่ชั่งได้ในช่วงเตรียมตรวจสอบรายการส่งออกยาง ซึ่งอาจต่ำกว่าน้ำหนักสุทธิหลังหักจากบิลเพราะยางสูญเสียน้ำ
_Avoid_: น้ำหนักสุทธิหลังหัก, น้ำหนักส่งออก

**ค่าทำงานต่อกิโลกรัม**:
อัตราค่าทำงานของรายการส่งออกยางต่อหนึ่งกิโลกรัมของน้ำหนักปัจจุบัน
_Avoid_: ราคายางเฉลี่ย, ค่าแรงรวม

**ค่าดำเนินการอื่น ๆ**:
ค่าใช้จ่ายเพิ่มเติมของรายการส่งออกยางที่นำมาบวกกับค่าทำงานตามน้ำหนัก
_Avoid_: รายการหัก, ส่วนลดค่าทำงาน

**ลบรายการส่งออกยาง**:
การยุติการใช้งานรายการส่งออกยางและปลดความสัมพันธ์ที่กันบิลซ้ำ โดยยังเก็บตัวรายการ ผู้ลบ และเวลาลบไว้เป็นประวัติ
_Avoid_: ลบถาวร, ยกเลิกรายงาน

**ข้อมูลบิล ณ เวลารวบรวม**:
รายละเอียดของบิลยางตามค่าที่ได้รับรองและล็อกอยู่ในเวลาที่ถูกนำเข้ารายการส่งออกยาง ใช้เป็นหลักฐานที่ไม่เปลี่ยนตามการแก้ไขบิลในอนาคต
_Avoid_: ข้อมูลบิลปัจจุบัน, สำเนาบิลใหม่
