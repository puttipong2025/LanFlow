# LanFlow

LanFlow จัดการงานการเงินและการปฏิบัติงานระหว่างสาขา โดยใช้คำศัพท์ร่วมกันเพื่อแยกความหมายทางธุรกิจที่ใกล้เคียงกันให้ชัดเจน

## การโยกเงินระหว่างสาขา

**การโยกเงินระหว่างสาขา**:
การนำเงินจากสาขาต้นทางไปยังสาขาปลายทาง โดยมีวิธีโยกเงินเป็นเงินสดหรือโอนธนาคาร
_Avoid_: โอนให้สาขา

**วิธีโยกเงิน**:
ช่องทางที่ใช้ส่งมอบเงินในการโยกเงินระหว่างสาขา มีสองแบบคือเงินสดและโอนธนาคาร
_Avoid_: ประเภทการโอน

**เงินสด**:
วิธีโยกเงินที่ส่งมอบเงินในรูปเหรียญและธนบัตรจริงจากสาขาต้นทางไปยังสาขาปลายทาง

**โอนธนาคาร**:
วิธีโยกเงินที่ส่งเงินผ่านบัญชีธนาคารและมีหลักฐานการโอน
_Avoid_: เงินโอน

**รอรับเงิน**:
สถานะของการโยกเงินสดที่สาขาต้นทางบันทึกการส่งเงินแล้ว แต่สาขาปลายทางยังไม่ได้ยืนยันว่าได้รับเงิน
_Avoid_: รอโอน, ค้างจ่าย

**ยืนยันรับเงิน**:
การที่สาขาปลายทางรับรองว่าได้รับเงินสดจากการโยกเงินระหว่างสาขาแล้ว
_Avoid_: อนุมัติรายการ

**จำนวนที่ส่ง**:
จำนวนเหรียญและธนบัตรที่สาขาต้นทางนับและส่งไปกับการโยกเงินสด

**จำนวนที่รับจริง**:
จำนวนเหรียญและธนบัตรที่สาขาปลายทางนับได้เมื่อรับการโยกเงินสด
_Avoid_: จำนวนที่รับ

**ผลต่างการรับเงิน**:
ส่วนต่างระหว่างจำนวนที่รับจริงกับจำนวนที่ส่ง แสดงได้ทั้งรายชนิดเงินและยอดรวม
_Avoid_: ยอดคงเหลือ

**ยอดไม่ตรง**:
สถานะการตรวจรับเงินสดที่จำนวนที่รับจริงไม่เท่ากับจำนวนที่ส่ง โดยต้องแสดงยอดผลต่างควบคู่กับสถานะ
_Avoid_: รับเงินแล้ว, เงินหาย

**ปิดยอดไม่ตรง**:
การที่ผู้มีสิทธิ์รับรองผลต่างการรับเงินและยุติการติดตามรายการ โดยคงจำนวนที่ส่งและจำนวนที่รับจริงเดิมไว้เป็นหลักฐาน
_Avoid_: แก้ยอด, อนุมัติยอด

**ลบถาวร**:
การนำรายการโยกเงินและข้อมูลการตรวจรับที่เกี่ยวข้องออกจากระบบโดยไม่สามารถกู้คืนได้ ทำได้เฉพาะ super admin
_Avoid_: ยกเลิก, ลบรายการ
